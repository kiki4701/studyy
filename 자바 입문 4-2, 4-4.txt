else if 를 쓸 때 이미 위의 조건절은 거짓인 상태가 된다.

그래서
int age = 8;

if(age <= 7){

	System.out.println("미취학") ;

} else if(age <= 13){

	System.out.println("초등학생");

}

할 경우 else if 절에 (age >= 8 && age <= 13)

이라고 들어가지 않아도 자동으로 age >= 8 이 참인 상태로 실행된다.
