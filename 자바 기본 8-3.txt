자바 기본 8-3

final은 변수의 값을 변경하지 못하게 막는다.

그러므로 기본형 변수의 경우 값을 변경할 수 없고, 참조형 변수의 경우 참조값을 변경할 수 없다.

만약 Data 라는 클래스 안에 public int value; 라는 변수가 있는 경우

final Data data = new Data(); 객체 생성을 한 뒤

data = new Data(); 이런식으로 같은 변수명으로 객체 생성을 또 하는 것은 불가능하다 (참조값을 다른 값으로 바꿀 순 없다)

하지만 data.value = 10; 이런식식으로 그 참조값 안의 변수 값을 변경 하는 것은 가능하다.

즉 final 이 붙으면 참조 대상 자체를 다른 대상으로 변경하지 못하는 것이지, 참조하는 대상의 값은 변경할 수 있다.(물론 참조값 안의 변수 값도 final 로 설정되어있으면 변경 불가, 여기서는 참조값 안의 변수는 public 이기때문에 가능한 것이다.)
