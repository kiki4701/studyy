웹을 개발하는 방법 3가지

-정적 컨텐츠
  서버에서 뭐 하는 것 없이 파일을 그대로 웹브라우저에 내려주는 것

-mvc와 템플릿 엔진 (가장 많이 하는 방식)
  html을 그냥 주는게 아니라 서버에서 프로그래밍 해서 html을 동적으로 바꿔서 내리      는 것

-API
  json 이라는 데이터 구조 포맷으로 클라이언트한테 데이터를 전달하는 방식
 

정적 컨텐츠는 스프링에서 기본적으로 제공한다

resources - static 폴더에 아무 html을 만들고 url 직접 입력하면 된다..

이때 동작과정은 내장 톰켓 서버가 요청을 받으면 스프링한테 hello-static.html 이라는 요청을 넘긴다

그 후 스프링은 컨트롤러에서 매핑된 것이 있는지 먼저 찾고 없으면 resources - static 폴더로 가서 같은 이름의 html 파일을 찾고 반환해준다
